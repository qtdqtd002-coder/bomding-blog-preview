## 1~5편 (확정 — 이번 연재분)

### 1편 — 게임 소개 + 쿠폰 + 시작 가이드  *(허브)*
- 제목(안): `템빨용사 쿠폰 번호 + 초보 시작 가이드 (가방=전투력 5분 오토배틀러)`
- 내용: 게임 정체성(5민랩, 가방 머지=전투력, 한 판 5분) / **공식 검증 쿠폰 현황**(있으면 번호+경로, 없으면 풀리는 채널 안내) / 우편함·사전예약 보상 / 첫 실행 동선·챕터 맛보기
- 핵심 키워드: `템빨용사 쿠폰`, `템빨용사 쿠폰번호`, `템빨용사 시작`
- 연결: 시리즈 출발점. 하단에서 "안 망하는 운영은 2편" 예고

### 2편 — 초보 꿀팁(초반 운영)  *(스포크)*
- 제목(안): `템빨용사 초보 꿀팁 — 초반에 이것만 알면 안 막혀요 (골드·훈련·가방 기초)`
- 내용: 챕터 구조(일반 9+보스 3) / 골드 사용처(상점 vs 훈련 3스탯) / 초반 훈련 우선순위 감각 / 가방 칸 관리 기초 / 흔한 초보 실수
- 핵심 키워드: `템빨용사 초보`, `템빨용사 꿀팁`, `템빨용사 초반`
- 연결: ← 1편(시작·쿠폰) / → 3편(누구 키울지)

### 3편 — 직업(영웅) 추천 티어 + 펫 육성  *(스포크)*
- 제목(안): `템빨용사 직업(영웅) 추천 티어 + 펫 육성 가이드`
- 내용: 직업 3특성축(치명타/지속피해/버프·디버프) / 공개 직업(바바리안·창병 등) / 펫 등급(신화) 성장 / "뭘 키우나" 방향
- 핵심 키워드: `템빨용사 직업 추천`, `템빨용사 티어`, `템빨용사 영웅`, `템빨용사 펫`
- 연결: ← 2편(초반 운영) / → 4편(그 직업에 맞는 조합)

### 4편 — 가방 배치 + 머지 조합  *(스포크, 시리즈 핵심)*
- 제목(안): `템빨용사 가방 배치 + 머지 조합 가이드 (시너지·세트로 전투력 올리기)`
- 내용: 가방=체스판 설계 / 머지 타이밍 / 시너지·세트 / 상점 4칸 선택 / 직업-아이템 매칭(창병=창 다수)
- 핵심 키워드: `템빨용사 가방`, `템빨용사 조합`, `템빨용사 머지`, `템빨용사 시너지`
- 연결: ← 3편(직업·펫) / → 5편(더 강해지는 콘텐츠 루프)

### 5편 — 중반 콘텐츠/재화  *(스포크)*
- 제목(안): `템빨용사 클랜·러시·오라 활용법 + 무과금 보석 모으기`
- 내용: 출시 탑재 콘텐츠 루프(러시→오라 조각→오라 강화 / 클랜 레이드→클랜 코인) / 재화 6종 우선순위 / 무과금 보석 동선 / 중반 정체 뚫기
- 핵심 키워드: `템빨용사 무과금`, `템빨용사 보석`, `템빨용사 클랜`, `템빨용사 오라`, `템빨용사 러시`
- 연결: ← 4편(조합 마스터)

## 6~12편 (후속 로드맵 — 검색수요 기반)

> ⚠️ 6편 이후는 출시 초기엔 미공개·미정립 정보(보스 패턴, 장비 수치, 펫 도감, 던전 등)가 많다.
> **각 편 작성 시점에 나무위키/공식/커뮤니티로 재조사**해 검증된 것만 쓴다(현 글로사리 기준 단정 금지 항목 다수).
> 메타가 무르익을수록 검색량이 커지므로, 정보 공개·메타 정립에 맞춰 순서·시기를 조정한다.

### 6편 — 챕터 보스 공략
- 제목(안): `템빨용사 보스 공략 — 챕터 보스 3라운드 뚫는 법 (정체 구간 돌파)`
- 내용: 챕터 = 일반 9 + 보스 3 구조 복습, 보스전 전 가방·시너지 점검 포인트, 막히는 챕터에서 뭘 보강하나
- 키워드: `템빨용사 보스`, `템빨용사 챕터`, `템빨용사 보스 공략`
- 연결: ← 4편(조합)·2편(초반)

### 7편 — 장비 심화(등급·강화·뽑기)
- 제목(안): `템빨용사 장비 가이드 — 등급(S영웅)·강화석·뽑기 천장 정리`
- 내용: 등급 체계(영웅/S영웅), 장비 강화석 운영, 뽑기 천장(영웅 10연·S영웅 50연 — 단일 출처 소프트 처리), 금 열쇠 효율
- 키워드: `템빨용사 장비`, `템빨용사 강화`, `템빨용사 뽑기`, `템빨용사 천장`
- 연결: ← 4편(조합)·5편(재화)

### 8편 — 펫 심화(등급·조합·펫 던전)
- 제목(안): `템빨용사 펫 추천 — 등급(신화·전설)·펫 던전·조합 가이드`
- 내용: 펫 등급(신화 0.05%·전설 2% — 소프트), 펫 알 효율, 펫 던전, 직업과 펫 시너지
- 키워드: `템빨용사 펫`, `템빨용사 펫 추천`, `템빨용사 펫 던전`
- 연결: ← 3편(육성). 3편이 허브, 8편이 펫 전용 스포크(키워드 분리)

### 9편 — 클랜 운영·클랜 레이드 심화
- 제목(안): `템빨용사 클랜 레이드 가이드 — 클랜 코인·시즌 랭킹 챙기기`
- 내용: 클랜 Lv.3+ 레이드, 1일 2회, 시즌 누적 데미지 랭킹, 클랜 코인 사용처, 클랜 고르는 법
- 키워드: `템빨용사 클랜`, `템빨용사 클랜 레이드`
- 연결: ← 5편(중반 콘텐츠). 5편이 개요, 9편이 클랜 전용 심화

### 10편 — 부가 콘텐츠 공략(도전의 탑·던전류)
- 제목(안): `템빨용사 도전의 탑·던전 공략 — 부가 콘텐츠로 재화 더 챙기기`
- 내용: 도전의 탑/펫 던전/용사 던전 등(인게임 한글 명칭 확정 후), 각 콘텐츠 보상·우선순위
- 키워드: `템빨용사 도전의 탑`, `템빨용사 던전`
- 연결: ← 5편(재화)·6편(보스)

### 11편 — 과금 효율·패키지 가성비
- 제목(안): `템빨용사 과금 가이드 — 패키지 가성비·무과금/소과금 우선순위`
- 내용: 패키지·상품 가성비 비교, 무과금/소과금 결제 우선순위, 보석 어디에 쓰나
- 키워드: `템빨용사 과금`, `템빨용사 패키지`, `템빨용사 가성비`
- 연결: ← 5편(무과금 보석). 무과금=5편, 과금효율=11편(키워드 분리)

### 12편 — 시즌·패치 대응(상시 갱신형)
- 제목(안): `템빨용사 업데이트 정리 — 신규 패치·시즌 콘텐츠 한눈에`
- 내용: 신규 패치노트·시즌 콘텐츠·신규 이벤트 정리(공지 나올 때마다 갱신), 복귀/신규 체크포인트
- 키워드: `템빨용사 업데이트`, `템빨용사 패치노트`, `템빨용사 이벤트`
- 연결: 전체 허브로 역링크(클러스터 마무리)
