메이크 드라마: MAD를 시작하셨다면 쿠폰부터 받고 가는 게 이득입니다. 지금 입력하면 일반 호출 티켓과 스태미나 비타민 같은 자원이 공짜로 들어오거든요. 2026년 6월 8일 기준으로 현재 살아 있는 쿠폰 코드와 입력 방법, 그리고 쓸 때 한 가지 꼭 주의할 점까지 정리했습니다.

📌 이 글 한눈에
  • 현재 유효 코드 3종 — WELCOMEMAD · MAD1STIN · MAKEVITA

📅 안내  아래 코드는 2026년 6월 8일 기준입니다. 쿠폰은 기간이 지나면 만료되고 새 코드가 수시로 추가되니, 입력 전에 유효기간을 한 번 확인하시는 걸 추천합니다.

시작하자마자 쿠폰부터 받아두면 초반 자원이 한결 넉넉해집니다.

이미지: 위메이드커넥트 / 게임메카

지금 쓸 수 있는 쿠폰 코드 (6월 8일 기준)

현재 입력 가능한 코드는 다음 3종입니다. 코드는 대문자 그대로 입력하면 되고, 보상은 우편함에서 받습니다.

쿠폰 코드보상기간
WELCOMEMAD일반 호출 티켓 10장상시(기간 미표기)
MAD1STIN일반 호출 티켓 10장~6월 30일
MAKEVITA스태미나 비타민 50개~6월 8일(오늘까지)

특히 'MAKEVITA'는 기간이 6월 8일까지라, 오늘 보신 김에 먼저 입력해두시는 걸 추천합니다. 'WELCOMEMAD'와 'MAD1STIN'을 합치면 호출 티켓만 20장이라, 초반 뽑기에 제법 보탬이 됩니다.

받은 보상은 어디에 쓰나

쿠폰으로 받은 보상이 어디에 쓰이는지도 짚고 가겠습니다. 일반 호출 티켓은 파트너를 뽑는 '호출'에 바로 쓰는 뽑기권이라, 초반 리세마라나 픽업 배너에 보탬이 됩니다. 스태미나 비타민은 '지역 제패' 같은 전투 콘텐츠에 입장할 때 소모하는 자원인데요. 비타민은 초반에 늘 모자란 편이라, 쿠폰으로 들어오는 50개가 생각보다 쏠쏠합니다. 둘 다 초반 진도에 직접 도움이 되는 보상이라, 받는 즉시 챙겨두면 됩니다.

쿠폰 입력 방법

처음이면 입력 메뉴가 어디 있는지 헷갈리는데, 경로만 알면 간단합니다.

단계방법
1게임 메인 화면에서 우측 상단 메뉴 열기
2공지사항으로 들어가 이벤트 탭 선택
3쿠폰입력에 코드를 정확히 입력
4우편함에서 보상 수령

시작 직후 뜨는 공지·이벤트 창에서도 같은 '쿠폰입력'을 찾을 수 있습니다. 이미 창을 닫았다면 위 경로로 다시 들어가면 됩니다.

• • •

쿠폰 쓸 때 꼭 알아둘 점

여기서 모르면 손해 보는 포인트가 하나 있습니다. 쿠폰 보상은 계정에 귀속된다는 점인데요. 즉 쿠폰을 입력한 뒤에 계정을 초기화하는 리세마라를 하면, 그 코드는 이미 쓴 것으로 처리돼 다시 쓸 수 없습니다. 그래서 리세마라를 돌릴 생각이라면 리세를 다 끝내고 정착한 계정에서 쿠폰을 입력하는 게 맞습니다.

다만 게스트 계정을 완전히 삭제하고 새로 시작하는 방식이면 쿠폰을 다시 쓸 수 있다는 이야기도 있으니, 본인 리세 방식에 맞춰 판단하시면 됩니다. 그리고 모든 쿠폰은 계정당 1회만 적용됩니다. 리세마라 자체가 처음이라면 소환·리세 순서부터 보고 오시는 게 좋습니다.

새 쿠폰은 어디서 확인하나

메이크 드라마: MAD는 출시 초기라 쿠폰이 자주 풀리는 편입니다. 출시 기념이나 인기 랭킹 달성 같은 이슈가 있을 때마다 새 코드가 나오는데요. 보통 공식 X(@MAkeDrama_KR)와 공식 카페 공지를 통해 올라오고, 기간 한정 깜짝 쿠폰도 수시로 등장합니다.

참고로 지난 주말에 풀렸던 'MADWEEKEND'(10만 크레딧) 같은 코드는 이미 기간이 끝나 입력해도 적용되지 않습니다. 이런 단기 코드는 놓치면 아쉬우니, 새 코드가 보일 때 바로바로 챙겨두시는 게 좋습니다. 이 글도 새 코드가 확인되면 갱신해두겠습니다.

자주 묻는 것 — 짧은 정리

쿠폰은 어디서 입력하나요?

우측 상단 메뉴에서 공지사항 → 이벤트 → 쿠폰입력 순으로 들어가면 됩니다. 보상은 우편함에서 받습니다.

쿠폰 입력하고 리세하면 다시 쓸 수 있나요?

계정 초기화 방식 리세라면 코드가 이미 사용 처리돼 다시 못 씁니다. 그래서 리세마라는 정착 후에 쿠폰을 넣는 게 안전합니다. 계정을 완전히 삭제하고 새로 시작하는 경우엔 재사용된다는 제보가 있습니다.

코드를 넣었는데 적용이 안 돼요.

대부분 기간 만료이거나 이미 사용한 코드입니다. 위 표의 유효기간을 확인하시고, 대소문자를 그대로 입력했는지도 한 번 점검해보세요.
