메이플 플래닛 2차·3차 전직은 레벨 30과 70에서 막히는 첫 관문인데, 알고 보면 핵심은 단순합니다. 2차는 '검은 구슬 30개', 3차는 '내 분신 격파 + 지혜의 목걸이 퀴즈'가 전부거든요. 직업별 교관 동선과 시험장 위치, 갓 30·갓 70 스펙으로 막히는 지점까지 한 편에 정리했습니다.

📌 이 글 한눈에
  • 2차(Lv30) — 1차 교관 ↔ 2차 교관 편지 심부름 → 시험장에서 검은 구슬 30개 → '영웅의 증거'로 전직
  • 3차(Lv70) — 엘나스 장로 관저 → 차원의 문에서 내 분신 격파 → 검은 부적 → 강인함의 목걸이 → 지혜의 목걸이 퀴즈 5문제
  • 전직 전 스킬포인트(SP)는 먼저 다 쓰고 가야 진행됨
  • 전사는 시험장 명중이 부족 → 명중 알약 챙기면 편함
  • 해적·시그너스 기사단은 절차가 조금 다름(본문 참고)
🔗 이어서 보면 좋은 글
아직 직업을 안 정했다면 → 메이플 플래닛 직업 추천 + 썬콜 공략 (무·과금 티어)

2차 전직 (Lv30) — '검은 구슬 30개'가 핵심

2차 전직은 퀘스트 창에 안 뜨고 NPC 대화로만 진행되는데요. 그래서 어디로 가야 할지 몰라 헤매는 경우가 많습니다. 흐름은 이렇습니다.

단계할 일
1Lv30 달성 → 1차 전직 시켜준 교관에게 말 걸기(대화 더보기) → "2차 교관에게 가라" 안내
2직업별 2차 교관을 찾아가 편지 받기 → 1차 교관에게 전달 → 답장을 다시 2차 교관에게 (왕복 심부름)
32차 교관이 시험장(전용 던전)으로 보내줌 → 안에서 몬스터 잡아 검은 구슬 30개 수집
4구슬 제출 → '영웅의 증거' 획득 → 2차 직업 분기 선택 후 전직 완료

2차 교관 위치와 갈라지는 직업은 아래와 같습니다. NPC 이름보다 '어느 마을 어느 쪽'인지로 기억하면 찾기 쉽습니다.

직업2차 교관 방향갈라지는 2차 직업
전사페리온에서 서쪽(서쪽 바위산 방면)파이터 / 페이지 / 스피어맨
마법사엘리니아 위쪽 숲 방면위자드(불·독) / 위자드(썬·콜) / 클레릭
궁수헤네시스 우측 던전 방면헌터 / 사수
도적커닝시티 우측 진행(페리온 가는 길목)어쌔신 / 시프
해적카이린에게 직접(왕복 심부름 없음)인파이터 / 건슬링거

해적은 다른 직업과 달리 카이린에게 곧장 시험을 받는 별도 흐름이고, 수집하는 증표 수량도 차이가 있습니다. 정확한 수량은 인게임에서 확인하시는 게 안전합니다.

전사 2차 동선인 페리온·서쪽 바위산 일대에서 만나는 파이어 보어. 갓 30이면 명중이 부족할 수 있습니다.

이미지: 메이플스토리 게임 자료(maplestory.io)

💡 전사 명중 팁  갓 30 전사는 시험장 몬스터에 공격이 자꾸 빗나가 구슬을 못 모으는 경우가 많습니다. 시험장 들어가기 전에 루디브리엄 잡화상점에서 명중 알약을 사 두면 한결 수월합니다.

3차 전직 (Lv70) — 분신을 이기고 퀴즈를 푼다

3차는 무대가 엘나스로 바뀝니다(엘리니아 아님 주의). 흐름은 '내 분신과의 대결 → 부적 → 퀴즈' 3단계입니다.

단계할 일
1Lv70 달성 → 엘나스 장로의 관저에 모인 직업별 3차 전직관에게 퀘스트 수령
2빅토리아로 돌아가 1차 전직관에게 → "네 분신을 쓰러뜨리고 부적을 가져오라" 지령
3차원의 문(수련 맵)에서 내 분신과 1:1 → 처치하면 검은 부적 획득
4검은 부적을 1차 전직관에게 제출 → 강인함의 목걸이로 교환 → 엘나스 3차 전직관에게 전달
5성스러운 돌(히든맵)에서 메이플 상식 퀴즈 5문제 → 다 맞히면 지혜의 목걸이 획득
6지혜의 목걸이를 3차 전직관에게 제출 → 3차 전직 완료

분신과 싸우는 차원의 문 위치는 직업마다 다릅니다. 갓 70이면 분신도 만만치 않으니 위치부터 정확히 알고 가는 게 좋습니다.

직업차원의 문(분신전) 위치갈라지는 3차 직업
전사개미굴 광장크루세이더 / 나이트 / 버서커
마법사사악한 기운의 숲 2메이지(불·독) / 메이지(썬·콜) / 프리스트
궁수슬리피 던전 5층레인저 / 스나이퍼
도적원숭이의 늪 2허밋 / 시프마스터
해적카이린 경유(전용 동선)버커니어 / 발키리

전사 차원의 문이 있는 개미굴, 궁수 슬리피 던전 일대에서 만나는 커즈아이.

이미지: 메이플스토리 게임 자료(maplestory.io)

💡 분신전 주의  분신은 봉인·암흑 같은 디버프를 겁니다. 만능 물약·성수·안약 등 상태이상 해제 수단을 꼭 챙겨 들어가세요. 그리고 퀴즈는 NPC·수치·드롭·퀘스트 같은 메이플 상식 5문제라, 모르면 미리 검색해 두면 한 번에 통과합니다.

흔히 막히는 부분

정리하면 막히는 지점은 거의 정해져 있습니다. ①스킬포인트(SP)를 안 쓰고 가는 것 — 2차·3차 모두 그동안 받은 SP를 다 써야 전직이 진행됩니다. ②전사 명중 부족 — 위에서 말한 명중 알약으로 해결. ③분신전 디버프에 그냥 들어가는 것 — 해제 물약 필수. ④3차를 엘리니아로 착각하는 것 — 무대는 '엘나스'입니다.

참고로 시그너스 기사단은 2026년 5월에 추가된 직업군이라 전직 절차가 모험가보다 간소한 것으로 알려져 있는데(나인하트·기사단장 경유), 플래닛에서의 세부 동선은 인게임·패치노트로 한 번 확인하시는 걸 권합니다.

한 줄 정리

2차(Lv30)는 교관 심부름 후 시험장에서 검은 구슬 30개 모아 '영웅의 증거'로 전직, 3차(Lv70)는 엘나스에서 받아 차원의 문 분신 격파 → 검은 부적 → 강인함의 목걸이 → 지혜의 목걸이 퀴즈 5문제로 마무리됩니다. 전직 전 SP 소진과 분신전 디버프 물약만 챙기면 막힐 일이 없습니다.

2차·3차만 넘으면 직업의 색깔이 본격적으로 드러나고, 레벨 120부터는 영웅의 별·펜타곤을 모으는 4차 전직이 기다립니다. 전직 단계별로 차근차근 밟아 나가시길 바라며, 2차·3차 전직 정리는 여기까지 하겠습니다.
