## 결론 (한 줄)
연봄은 현재 **애드센스 "저가치 콘텐츠" 거절 리스크가 낮다.** 4축 중 ②저가치 집계·③얇은 글·④본문 광고 과다는 **위험 없음**. 단 **①cookie-cutter(균일 템플릿)만 황색 경보** — 알맹이는 글마다 다르나 껍데기(골격)가 거의 100% 동일해, 발행량이 더 쌓이면 "템플릿 양산"으로 비칠 여지가 있다. 굳히면 안 되는 패턴을 아래 예방 항목에 정리.

## ① cookie-cutter 상세 — 가장 균일한 패턴

전 클러스터(챔피언스·GO·30주년·애니)가 공유하는 **단일 골격**:

```
도입 문단 → [한눈에 보기] 박스 → 마지막 업데이트 줄 →
(섹션 칩 라벨 + h2 + 본문 + 표/이미지 + 💙연/🩷봄 한마디) × 6회 →
자주 묻는 질문(FAQ Q칩 4개) → 🩷봄의 한마디로 정리 → 태그칩 → 구독배너
```

- **h2 정확히 6개 고정**: 30주년·애니 클러스터를 뺀 거의 모든 글이 h2:6. 섹션 수가 콘텐츠 필요가 아니라 틀에 맞춰져 있을 가능성.
- **h3 미사용**: 챔피언스·GO 클러스터 전 글 h3:0 (애니·아틀리에만 h3 사용) → 본문 위계가 평평.
- **자동 메타 함정 주의(학습 반영)**: 티스토리는 og:description을 도입 문장으로 자동 생성하므로, 도입부까지 같은 톤("~정리해 봤어요/짚어 볼게요")이면 스니펫도 균일해 보인다. 메타 '전무' 오진과 별개로, **도입 첫 문장 패턴 다양화**가 스니펫 차별화에 유효.

> 이게 위험인 이유: 애드센스의 cookie-cutter 판정은 "여러 글이 같은 틀로 찍어낸 듯"한 인상에 반응한다. 연봄은 알맹이(데이터·해석)가 달라 지금은 통과 가능성이 높지만, 같은 6-h2 + 연/봄 한마디 + FAQ 골격이 50편·100편으로 누적되면 패턴 자체가 시그널이 된다.

## 균일도 황색이지만 거절급은 아닌 근거
- 골격은 같아도 **본문은 매 글 고유 데이터**(상성 배수·VP 경제·시세표·드니차 CD 일정 등)로 채워짐.
- 매 글 **자체 해석·오해 정정·실전 우선순위**가 들어가 "긁어 붙인 집계"와 구분됨.
- 분량·구조가 thin이 아님. → 종합하면 "양산 인상 리스크"는 있으나 "내용 빈약" 거절 사유엔 해당 안 함.

## GSC 실측 교차 (저가치 진단 보강)

- 기간 합계: clicks 541 / impr 2,729 / CTR 19.8% / 평균순위 5.73 (직전 동일기간 0 = 신생, 06-11 발행 개시).
- **유입 극단 집중**: /11(챔피언스 리세마라)이 499클릭으로 **전체의 92%**. 쿼리 "포챔스 리세"(CTR 47.8%·1.81위) 등 챔피언스 리세마라 계열이 효자.
- 나머지 33편은 0~15클릭. 노출은 있으나 클릭 0인 글 다수(/2·/14·/12 등) → **저품질이 아니라 색인·노출 초기 단계**(신생 + 경쟁). thin 때문이 아님.
- CTR 19.8%·평균순위 5.73은 신생 블로그로 양호 → 검색엔진이 저품질로 디스카운트하는 신호 없음.

## 예방 관점 — "굳히면 안 되는 패턴" (신생이라 지금이 적기)

1. **6-h2 + 연/봄 한마디 + FAQ 골격을 모든 글에 기계적으로 반복하지 말 것.** 구조 프로파일 다변화(G/N/R/H/D)를 실제로 글마다 흔들어, h2 개수·섹션 칩 유무·FAQ 유무를 콘텐츠에 맞게 가변화. (정본: structure-profile-diversification — 연봄 적용 대상)
2. **도입 첫 문장 템플릿("~정리해 봤어요/짚어 볼게요") 다양화.** 자동 메타 스니펫이 균일해지는 직접 원인.
3. **순수 집계형 글(시세표만·상성표만)을 해석 없이 양산하지 말 것.** 현재는 해석이 동반돼 안전 — 이 습관을 유지하는 게 방어선.
4. **애드센스 승인 후 본문 광고 배치 상한 관리.** 본문 길이 2,600~6,500자 대비 광고 밀도가 과해지지 않게(문단 사이 무한 삽입 금지). 현재 0이라 무위험이나 승인 직후가 함정 구간.
5. **효자글 단일 의존(/11 92%) 완화.** 저가치 이슈는 아니나, 다른 글의 색인·노출이 붙도록 내부링크·리프레시. (분석 범위 밖이지만 분산이 블로그 건강에 유리)

## 측정 한계 (솔직히)
- 광고 0은 로컬·라이브 둘 다 확인했으나 **애드센스 미승인 시점이라 광고 자체가 없는 게 정상** — "광고 과다 아님"은 현 시점 사실이고, 승인 후엔 재측정 필요.
- 본문 자수는 태그 제거 후 공백제외 추정치(라이브 렌더 기준과 ±오차).
- URL 번호↔제목 매핑은 /11(리세마라)·/4(띠부씰 100종)·/8(챔피언스 입문) 등 본문 내부링크로 일부만 확정. 나머지 번호↔제목은 미확정("확인 불가").
- 색인 수는 GSC sitemaps의 indexed 필드(폐기)로 판단하지 않음 — 본 진단은 노출·클릭 실측과 본문 품질로만 저가치 여부를 판정.
