힐링게임추천 더코지플로리스트 초반가이드 쿠폰코드 정리 2026년6월최신

봄딩2026년 6월 4일· 게임

힐링 게임 찾으시는 분들, 오늘 딱 하나 소개해 드릴게요. 게임 하다가 진짜 꽃다발이 집으로 배송된다는 말, 들어보셨나요? 처음에 저도 반신반의했는데, 이게 실제로 존재하는 시스템이더라고요. 바로 더 코지 플로리스트(The Cozy Florist)라는 게임인데요, 올해 5월에 한국에도 정식 출시되면서 글로벌에서 이미 1,000만 다운로드를 넘긴 힐링 SNG입니다. 오늘은 처음 시작하는 분들을 위해 초반 가이드랑 **지금 바로 쓸 수 있는 쿠폰 코드**까지 한 번에 정리해 드릴게요.

더 코지 플로리스트 — 한국 구글플레이에서 바로 설치 가능하답니다

이미지 출처: 구글플레이 더 코지 플로리스트 · 네이버 본문엔 받아서 재업로드 권장

더 코지 플로리스트가 뭔 게임이에요?

한마디로 정리하면, 나만의 꽃집을 운영하는 힐링 SNG인데요, 개발사 RiftSky Games에서 만들었고 이미 북미·유럽 가드닝 카테고리에서 1위를 찍은 게임이에요. 한국에는 2026년 5월에 정식 출시됐고, Android랑 iOS 모두 지원하니까 어느 기기든 상관없이 즐길 수 있답니다.

이 게임이 특이한 건 '리얼 배송 시스템'이라는 거예요. 게임 안에서 꽃을 재배하고 수확해서 '꽃 인사 카드'를 모으면, 실제 꽃다발을 배송받을 수 있는 이벤트에 참여할 수 있는데요, 그냥 화면 속 꽃으로 끝나는 게 아니라는 점이 다른 힐링 게임들이랑 차별화되는 포인트죠. 실패 패널티가 없고 방치형으로 운영되기 때문에, 바쁜 일상 중에 잠깐씩 들어와서 꽃이나 살펴보는 느낌으로 즐기기 딱 좋은 게임이에요.

초반에 이렇게 돌아가요

📷스샷 1: 정원에서 꽃을 재배 중인 화면 (성장 타이머 UI 포함)
📸 직접 촬영 — 게임 내 정원 탭 / 씨앗이 자라는 중인 화면, 타이머가 보이게 / 참고: The Cozy Florist 워크스루 영상

씨앗 심고, 기다리고, 수확하고 — 이 루프가 기본이에요

기본 흐름은 간단해요. 씨앗 심기 → 재배(대기) → 수확 → 부케 조합 → 손님 주문 이행 순서로 진행되는데요, 방치형이라 꽃이 자라는 건 오프라인 중에도 계속되니까 자주 들어와서 수확하고 다시 심어두는 패턴으로 하면 됩니다. 처음엔 데이지나 백합 같은 기본 꽃부터 시작하고, 레벨이 오르면서 희귀한 꽃들로 점점 확장되는 구조라 수집욕을 자극하더라고요.

재화는 크게 셋으로 나뉘는데, 골드(Gold)는 기본 화폐로 씨앗 구매나 정원 업그레이드에 쓰이고, 다이아몬드는 프리미엄 화폐라 희귀 아이템이나 가속에 아껴두는 게 좋아요. 가속카드는 재배 타이머를 단축할 때 쓰이는데, 희귀 꽃일수록 기다리는 시간이 길어지니 전략적으로 사용하는 것이 좋습니다. 처음 시작하면 '신규 씨앗 메가팩'을 받을 수 있으니까, 접속하자마자 우편함 먼저 챙겨주시는 걸 추천드리는데요.

초반에 이것만 챙겨요!

📷스샷 2: 신규 보상 또는 우편함 수령 화면
📸 직접 촬영 — 게임 접속 후 우편함 탭 / 신규 씨앗 메가팩 또는 보상 팝업 화면

처음 접속하면 꼭 우편함 확인하세요

시작할 때 도움 되는 것들 간단하게 짚어볼게요. 우선 쿠폰 코드를 가장 먼저 입력하는 것이 좋아요. 코드 입력 경로는 아래에 안내해 두었으니 꿀팁 바로 쓰세요. 그다음으로는 다이아몬드를 골드로 살 수 있는 곳에 낭비하지 않는 것이 중요한데요, 초반에 다이아가 아까워서 이것저것 쓰다 보면 나중에 진짜 필요할 때 없어서 아쉬운 경우가 생기더라고요.

또 친구 정원 방문을 적극 활용하는 것도 추천드리는데요, 다 자란 꽃을 가져올 수 있는 '꽃 가져오기' 기능이 있어서 희귀 씨앗 확보에 도움이 됩니다. 마지막으로 일일 미션이랑 광고 시청으로 무료 재화를 꾸준히 챙기면 과금 없이도 어느 정도 성장이 되니까, 매일 들어올 때 미션 탭 한 번씩 확인해 주는 게 좋아요.

쿠폰 코드 정리 (2026년 6월 최신)

📷스샷 3: 교환 코드 입력 메뉴 화면 (프로필→설정→교환 코드)
📸 직접 촬영 — 프로필 아이콘 탭 → 설정 → '교환 코드' 입력창이 보이는 화면

이 화면이 코드 입력하는 곳이에요 — 대소문자 구분 있으니 정확하게!

코드 입력 방법은 간단해요. 게임 메인 화면에서 프로필 아이콘 → 설정 → 교환 코드로 들어가서 입력하면 되고, 보상은 우편함에서 받으면 됩니다. 대소문자를 정확히 구분하니까 그 점만 주의해 주세요.

지금 챙길 수 있는 코드부터 먼저 알려드릴게요. 공효진 배우와 함께하는 공식 이벤트 코드 4종이 2026년 7월 31일까지 유효한데요, 공식 인스타그램(@thecozyfloristkr) 이벤트 게시물에서 응원 댓글 달성 수에 따라 순차적으로 해금되는 코드라, 현재 어느 단계까지 열렸는지는 공식 인스타에서 확인하는 것이 가장 정확하답니다.

여기서 끝이 아닌 게, 매주 공식 인스타그램(@thecozyfloristkr)에서 '정원 주말 기프트 코드'라는 이름으로 주간 코드가 새로 나와요. 유효 기간이 약 1주일 정도라 빠르게 사라지는 코드들이라서, 공식 인스타 팔로우해 두고 주기적으로 확인하는 게 가장 확실하게 챙기는 방법이에요. 공식 페이스북 그룹에도 같이 올라오니까 참고하시면 되겠습니다.

네이버에서 즐기는 분들은 공식 라운지(game.naver.com/lounge/Flower_Town_Story/home)에서 공지랑 이벤트 소식을 확인할 수 있어요. 인스타그램을 주로 쓰신다면 @thecozyfloristkr를 팔로우해 두시면 주간 코드가 게시물로 올라올 때 바로 알림이 가니까 편리하고요. 카카오톡 오픈채팅(open.kakao.com/o/p7Ym49ni)도 운영 중이라 유저들끼리 정보 공유도 활발하게 이루어지고 있답니다.

더 코지 플로리스트 정리

간단하게 정리해 보면, 더 코지 플로리스트는 2026년 5월 한국 정식 출시된 힐링 SNG로, 꽃 재배와 꽃집 운영을 즐기면서 꽃 인사 카드를 모으면 리얼 꽃다발 배송 이벤트에도 참여할 수 있는 게임이에요. 실패 패널티 없는 방치형 구조라 부담 없이 즐길 수 있고, 현재 공효진 이벤트 코드 4종(KongHyoJin0404 외 3종, 7월 31일까지)을 비롯해 다양한 쿠폰 코드가 배포 중이니 지금 시작하면 초반 재화 챙기기 딱 좋은 타이밍이에요.

힐링 게임 찾고 계셨다면 한 번 깔아보시면 좋을 것 같아요. 꽃집 꾸미는 느낌도 나고, 뭔가 채워지는 재미가 있어서 슬슬 손이 가는 게임이더라고요ㅎ
