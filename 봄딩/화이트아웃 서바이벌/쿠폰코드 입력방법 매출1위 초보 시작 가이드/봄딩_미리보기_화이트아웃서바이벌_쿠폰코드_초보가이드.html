모바일 게임 매출 순위 한 번씩 들여다보는 분들이라면 요즘 이 이름 안 보셨을 리가 없는데요. 바로 화이트아웃 서바이벌(Whiteout Survival)이에요. 빙하기가 와버린 세상에서 살아남는 생존 전략 게임인데, 영문 약자로 WOS라고들 부르더라고요. 오늘은 이 게임을 막 시작하시는 분들이 제일 궁금해하는 두 가지, 쿠폰코드 입력 방법이랑 초반에 뭐부터 손대면 좋은지를 한 번에 정리해볼게요.

눈 덮인 세상에서 도시 세우고 살아남는 생존 전략 게임이에요.

5월 매출 1위, 그 화이트아웃 서바이벌 맞아요

"생존 게임이 그렇게 잘 나가?" 싶으실 텐데, 수치를 보면 좀 놀라워요. 아이지에이웍스 모바일인덱스가 집계한 2026년 5월 한국 모바일 게임 매출 순위에서 화이트아웃 서바이벌이 약 331억 원으로 1위를 찍었거든요. 메이플 키우기(285억), 킹샷(271억), 리니지M(223억) 같은 쟁쟁한 게임들을 다 제친 결과라 더 눈에 띄더라고요.

매출이 꾸준히 받쳐주는 게임은 그만큼 운영도 부지런하다는 뜻이라, 쿠폰(기프트 코드)도 꽤 자주 풀리는 편이에요. 이벤트나 업데이트, 기념일 같은 때 슬쩍슬쩍 올라오니까 한 번 입력 방법만 익혀두면 두고두고 챙겨 먹기 좋더라고요.

쿠폰코드 입력 방법부터 (게임 안 / 웹 둘 다)

이게 사실 제일 중요해요. 코드 자체보다 입력 경로를 모르면 코드가 있어도 못 쓰거든요. 화이트아웃 서바이벌은 코드 입력하는 길이 두 가지예요.

① 게임 안에서 입력 (안드로이드)
게임 접속 → 화면 왼쪽 위 내 아바타(프로필)를 눌러요 → 프로필 창이 뜨면 오른쪽 아래 설정(톱니바퀴) 아이콘을 누르고 → 거기서 '기프트 코드(Gift Code)' 버튼을 찾아 들어가면 입력창이 나와요. 거기에 코드를 하나씩 넣고 받으면 됩니다.

② 웹사이트에서 입력 (아이폰 포함 공용)
아이폰은 보통 게임 안 입력 대신 공식 코드 교환 사이트를 써요. wos-giftcode.centurygame.com(센추리게임 공식 페이지)에 들어가서 → 게임 설정에서 확인한 플레이어 ID로 로그인하고 → 코드와 그 아래 인증 코드(캡차)를 입력한 뒤 → '확인'을 누르면 게임 우편함으로 보상이 들어옵니다.

📷스샷 1: 아바타 → 설정(톱니바퀴) → '기프트 코드' 버튼이 보이는 화면
📸 직접 촬영 권장 — 게임 실행 후 아바타→설정 들어가서 '기프트 코드' 버튼 위치가 보이게 찍어주시면 독자분들이 바로 따라오기 좋아요.

아바타 눌러서 설정 들어가면 기프트 코드 버튼이 거기 숨어 있어요.

주의할 점도 두 가지만 짚을게요. 코드는 대소문자를 정확히 구분하니까 보이는 그대로 옮겨 적어야 하고요, 계정당 한 번씩만 사용되는 데다 유효기간이 있어서 금방 만료돼요. 그래서 보이면 바로 입력하는 게 제일 안전해요.

최근 보고된 쿠폰코드 후보 (만료 가능 · 꼭 직접 확인하고 쓰세요)

여기서 제가 솔직하게 말씀드릴 부분이 있어요. 화이트아웃 서바이벌 코드는 정말 자주 바뀌고, 며칠이면 만료돼버려요. 코드 정리해주는 해외 사이트들끼리도 "이건 살아있다 / 이건 죽었다"가 서로 갈릴 정도라, 제가 "이 코드 무조건 됩니다" 하고 단정 짓는 게 오히려 위험하더라고요. 그래서 아래 코드는 최근(6월 21일경) 살아있다고 보고된 후보로만 봐주시고, 꼭 위 방법으로 직접 넣어 확인해 주세요. 이미 만료됐을 수도 있어요.

코드를 넣으면 보통 다이아(젬), 자원(고기·나무·석탄·철), 가속 아이템 같은 게 우편함으로 들어와요. 초반에 자원 한 줌이 은근히 크니까, 되는 코드만 골라 챙겨도 시작이 한결 가벼워집니다.

그리고 가장 확실한 방법은요, 최신 코드를 공식 채널에서 직접 보는 것이에요. 화이트아웃 서바이벌은 공식 디스코드 · 공식 페이스북 · 게임 내 우편 공지로 새 코드를 풀거든요. 새 코드는 거기서 그때그때 확인하시는 게 제일 정확해요. 저도 쓸 만한 코드가 새로 풀리면 챙겨서 업데이트해 둘게요.

초보 시작 가이드 - 뭐부터 손대면 될까요?

이제 막 깔았다면 어디부터 만져야 할지 막막하시죠. 이 게임은 도시(기지)를 키우면서 영웅을 모으고 연맹에 들어가 같이 굴러가는 구조라, 큰 줄기만 잡으면 의외로 길이 보여요. 초반 우선순위를 흐름대로 정리해볼게요.

우선 메인 퀘스트(주요 임무)를 따라가세요. 이 게임은 초반에 메인 임무가 시키는 대로만 해도 건물 짓는 순서, 영웅 뽑는 타이밍, 자원 쓰는 법을 자연스럽게 익히게 돼 있어요. 임무 보상으로 받는 자원이랑 가속이 또 쏠쏠해서, 처음엔 임무 줄기를 등대 삼아 따라가는 게 제일 안전합니다.

그다음은 핵심 건물부터 올리는 거예요. 모든 걸 동시에 올리려 하지 말고, 중심이 되는 본부(중심 건물인 '화로') 레벨을 우선 올리는 데 집중하면 다른 건물·연구의 상한도 같이 풀려요. 자원은 늘 부족하게 느껴지니까, 욕심내서 여기저기 벌리기보다 한 줄기로 쭉 올리는 게 초반엔 더 빨라요.

📷스샷 2: 초반 도시 화면(건물 배치)과 메인 임무 안내가 보이는 장면
📸 직접 촬영 권장 — 초반 도시 전경 + 화면 한쪽의 메인 임무 표시가 같이 나오게 찍으면 "이거 따라가면 되는구나"가 한눈에 와요.

처음엔 이 임무 줄기만 따라가도 절반은 먹고 들어가요.

영웅은 초반에 주는 걸로 충분해요. 시작하면서 얻는 영웅들과 임무·이벤트로 주는 영웅을 우선 키우면 되고요, 무리해서 처음부터 뽑기에 매달릴 필요는 없어요. 받은 영웅을 골고루 성장시키면서 전투/생산 라인을 채우는 쪽이 초반엔 더 체감이 좋더라고요.

그리고 연맹(Alliance)은 되도록 빨리 가입하세요. 이게 초보 분들이 제일 늦게 챙기는데, 사실 제일 빨리 했어야 하는 것 중 하나예요. 연맹에 들어가면 건설 도움(가속)·연맹 상점·각종 지원을 받을 수 있어서 성장 속도가 확 달라지거든요. 활동적인 연맹 하나 잘 골라 들어가는 것만으로도 혼자 할 때보다 훨씬 수월해집니다.

정리하면 이래요. 메인 임무 따라가기 → 핵심 건물 한 줄기로 올리기 → 받은 영웅 키우기 → 연맹 빨리 가입. 이 네 가지 흐름만 잡아도 초반은 헤매지 않고 갈 수 있어요. 세세한 빌드나 영웅 조합은 게임이 어느 정도 익은 뒤에 천천히 파셔도 늦지 않아요.

정리하며

매출 1위라는 타이틀이 괜히 붙은 게 아닐 만큼 콘텐츠가 두툼한 게임이라, 쿠폰으로 초반 자원 챙기고 + 메인 임무·연맹으로 길 잡고 시작하면 입문 장벽이 꽤 낮아져요. 쿠폰은 워낙 수시로 바뀌는 정보라 새 코드는 위에 적은 공식 채널에서 그때그때 확인하시는 걸 꼭 권해드리고요, 보이는 코드는 만료되기 전에 미리미리 챙겨두세요. 빙하기 세상에서 도시 하나 야무지게 키워보시면 좋겠네요!
