일랜시아 리마스터 Project ER 공식 티저 공개 NEXON Replay 프리뷰

봄딩2026. 6. 5.· 게임

2025년 12월 18일, 넥슨이 오래된 클래식 IP들을 외부 창작자에게 개방한다는 'NEXON Replay' 프로젝트를 발표했는데요. 그 5종 IP 중에 일랜시아가 포함되면서 꽤 많은 분들이 반응하셨더라고요. 추억의 그 게임이 다시 살아날 수도 있겠구나 싶었던 거죠. 오늘은 외부 개발팀이 공식 티저를 공개한 Project ER이 어떤 프로젝트인지 정리해 드릴게요.

Project ER 공식 티저 — 그리운 일랜시아가 돌아옵니다.

이미지: Project ER 공식 유튜브 채널 (@project_er) · 네이버 본문엔 받아서 재업로드 권장

NEXON Replay가 뭔 프로젝트예요?

넥슨이 2025년 12월 18일 발표한 NEXON Replay는, 자사의 클래식 IP를 외부 개발자·창작자에게 라이선스로 개방하는 프로젝트입니다. 개방된 IP는 총 5종으로, 어둠의전설·일랜시아·아스가르드·택티컬 커맨더스·에버플래닛이에요.

중요한 점은 넥슨이 직접 리마스터를 만드는 게 아니라, 외부 개발팀이 넥슨 IP 라이선스를 받아 독자적으로 개발하는 구조라는 거예요. 그래서 같은 일랜시아 IP를 여러 팀이 서로 다른 방향으로 개발할 수 있는 비독점 개방형 방식이고요. Project ER은 그 중에서 원작 감성을 살린 MMORPG 트랙을 개발 중인 팀의 프로젝트 이름입니다.

NEXON Replay 공식 사이트에서 소개 중인 개방 IP들이에요.

이미지: NEXON Replay 공식 사이트 (iphub.nexon.com) · 네이버 본문엔 받아서 재업로드 권장

일랜시아, 어떤 게임이었죠?

일랜시아는 1999년 정식 서비스를 시작한 넥슨의 2D MMORPG로, 바람의나라·어둠의전설·아스가르드·테일즈위버와 함께 흔히 넥슨 클래식 5인방으로 불리던 게임이에요. 1998년 베타를 거쳐 1999년에 정식 오픈했는데, 2000년대 초 PC방을 달리던 분들이라면 한 번쯤 이름은 들어봤을 거예요.

원작 일랜시아의 인게임 화면 — 이 감성, 기억하시나요?

이미지: 넥슨 공식 일랜시아 홈페이지 · 네이버 본문엔 받아서 재업로드 권장

일랜시아의 가장 독특한 점은 '레벨'이 없다는 거예요. 대신 어빌리티(0.00~100.00) 시스템으로 성장하는데, 검을 쓰면 검 어빌리티가, 낚시를 하면 낚시 어빌리티가 오르는 방식이에요. 내가 하고 싶은 걸 하면 자연스럽게 강해지는 구조라서, 레벨 압박 없이 진짜 자유롭게 놀 수 있었던 게임이었죠.

생활형 콘텐츠도 굉장히 다양했는데요, 요리·재단·세공·미용·낚시·광물발굴 같은 것들을 전투 없이도 즐길 수 있었어요. 그래서 "게임 안에서 살았다"는 표현이 딱 맞는 분들이 많더라고요. 거기다 지역마다 다른 BGM이 있었는데, 포프레스네나 세르니카 음악만 들어도 당시 기억이 확 떠오른다는 분들이 아직도 많아요. 사람들이랑 어울리고, 인연 맺고, 그냥 거기 있는 게 좋았던 게임이랄까요 ㅎ.

그 시절 클래식 MMORPG 감성이 그대로예요.

이미지: 넥슨 공식 일랜시아 홈페이지 · 네이버 본문엔 받아서 재업로드 권장

Project ER 공식 티저 이렇게 나왔어요!

Project ER 팀이 공식 유튜브 채널(@project_er)에 공식 티저 영상을 올렸는데요, 영상 자체는 약 1분 남짓의 짧은 구성이에요. 원작 일랜시아 소개 컷으로 시작해서 NEXON Replay 프로젝트 소개로 이어지고, 중간에 Project ER 타이틀 컷과 함께 인게임 티저 장면 두 컷이 공개됩니다. 인게임 장면을 직접 보면 원작의 2D 횡스크롤 감성이 어느 정도 유지된 느낌이 들어요.

아직 가제(가칭)로 운영 중인 팀이라 세부 정보는 공개가 제한적인데요, CBT(비공개 테스트)는 2026년 9월경 진행될 예정이라고 해요. 현재 디스코드 채널도 운영 중이라(discord.gg/elancia) 개발 소식을 가장 빠르게 받아보실 수 있을 것 같습니다.

기대도 되고, 걱정도 되고

추억의 게임이 다시 나온다는 소식에 많은 분들이 반겼는데요, 기대와 함께 걱정도 함께 나오더라고요. 긍정적인 반응으로는 역시 원작의 어빌리티 시스템과 생활형 자유도가 현대에 어떻게 살아날지 기대가 크고, 그때 그 BGM 감성을 다시 느낄 수 있을지에 대한 기대도 많아요.

반면 우려 섞인 시각도 있는데요, "요즘 게임 기준에서 불편한 게 많을 것"이라는 걱정이나, 과거 원작에서 문제였던 오토·매크로 관련 우려도 여전히 있어요. 또 NEXON Replay 프로젝트 자체가 아직 초기 단계라 모든 팀이 무사히 개발을 완주할지에 대한 불안감도 공존하는 편이고요. 기대는 크지만 지켜봐야 한다는 분위기라고 보면 맞을 것 같네요.

일랜시아 리마스터 정리

Project ER과 일랜시아 리마스터 소식을 정리해 보면, 넥슨이 2025년 12월 18일 NEXON Replay 프로젝트를 통해 일랜시아 IP를 외부 개발팀에 개방했고, Project ER 팀이 그 라이선스를 받아 공식 티저까지 공개한 상태예요. 아직 CBT 일정(2026년 9월 예정)만 나온 상황이라 사전예약이나 추가 정보는 없고, 개발이 어떻게 진행될지 조금 더 지켜봐야 할 것 같습니다.

그래도 오랫동안 잠들어 있던 일랜시아가 다시 이야기되고 있다는 것 자체가 반가운 분들 많으실 것 같아요. 공식 채널(@project_er)이나 디스코드에서 소식 챙겨두시면 좋겠습니다.
