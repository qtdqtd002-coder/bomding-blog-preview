1.1 '꿈의 회랑' 업데이트로 추가된 해오름섬 (NTE)

0. 시작 전 — 이것 먼저 알고 가세요

괴담집은 총 9개이고, 그냥 필드를 걷다 줍는 수집품이 아닙니다. 퍼즐 해제나 특정 조건을 충족해야 생기는 방식이라 모르면 섬 다 뒤져도 하나도 못 먹는 경우가 생겨요.

제일 중요한 포인트가 하나 있는데요. 괴담집 중 절반 이상이 '밤 시간대' 조건을 요구합니다. 이환에는 시간 조작 기능이 있으니, 낮에 돌아다니다가 조건이 밤이면 바로 시간 바꿔서 클리어하시면 됩니다. 모른다면 설정 메뉴에서 시간 조작 기능을 먼저 찾아두세요. 이거 모르고 밤 될 때까지 기다리면 시간 낭비입니다(ㅋㅋ).

그리고 해오름섬은 메인 스토리 어느 정도 진행 후 해금됩니다. 아직 섬이 안 열렸다면 메인 스토리를 더 진행해 주세요.🔍 정확한 선행 챕터 인게임 확인

📌 공략 전 핵심 요약
· 괴담집 총 9개
· 밤 시간 조건 多 → 시간 조작 기능 미리 숙지
· 일반 필드 수집과 달리 퍼즐·행동 조건 충족 필요
· 발자국 포인트 100% 달성하려면 전부 수집 필요

1. 해변의 수호신

한 줄 조건 — 해변 NPC에게 대화 후 스태미너 소진까지 헤엄치기

해오름섬 해변 쪽에 있는 특정 NPC와 먼저 대화를 나눠주세요. 대화를 마치면 바다에 뛰어들어서 스태미너가 다 떨어질 때까지 헤엄을 치면 됩니다. 억지로 수영하다 익사(?) 직전까지 가면 클리어된다는 걸 기억해두세요. 단순하지만 모르면 그냥 지나치기 쉬운 유형이에요.

📷스샷 1: 해변 NPC 대화 화면 + 수영 스태미너 게이지
📸 직접 촬영 권장 — 스태미너 바 거의 다 닳은 순간 + NPC 위치를 함께 찍어두면 독자에게 위치 전달 효율 높음. 참고: 인벤 괴담집 전체 공략

스태미너 다 닳을 때까지 헤엄치면 클리어!

2. 유령 터널

한 줄 조건 — 비 오는 날씨 + 밤 시간대에 터널을 걸어서 통과

조건이 두 개라 좀 까다롭게 느껴지는데요. 날씨가 비인 상태에서 밤 시간에 해오름섬 터널을 걸어서 지나가면 됩니다. 날씨 조작이 가능하다면 비로 맞춰두고, 시간도 밤으로 바꾼 다음 터널 입구에서 쭉 걸어서 통과하세요. 뛰지 말고 걸어서 통과해야 한다는 점 주의하세요.🔍 뛰기 불가 조건 인게임 확인

📷스샷 2: 비 오는 밤 + 터널 내부 화면
📸 직접 촬영 권장 — 터널 안쪽에서 비ㆍ밤 날씨 조건 동시에 보이는 화면. 참고: 인벤 괴담집 공략

비 + 밤, 두 조건 맞추고 터널 걸어서 통과

3. 낚시터

한 줄 조건 — 낚시로 '푸른 비늘 깃발' 아이템 획득

해오름섬 낚시터에서 낚시를 하다 보면 '푸른 비늘 깃발'이라는 특정 아이템이 잡히는데요. 이게 나올 때까지 낚시를 반복해야 합니다. 드랍이 확정이 아니라 랜덤이라 몇 번 캐스팅해야 할 수도 있는데, 꾸준히 낚다 보면 나옵니다. 낚시 스팟은 섬 내 낚시 포인트 어디서나 가능합니다.🔍 낚시 스팟 한정 여부 인게임 확인

📷스샷 3: 낚시터 화면 + '푸른 비늘 깃발' 획득 팝업
📸 직접 촬영 권장 — 아이템 획득 팝업이 뜬 순간 캡처.

'푸른 비늘 깃발' 획득하면 괴담집 클리어

4. 용 같이

한 줄 조건 — 유리병 발견 → 부싯돌 획득 → 모닥불 불 붙이기 → 탈옥수 NPC → 암호 '다루가' 선택

단계가 조금 있는 편이라 순서대로 따라가시면 됩니다.

① 해오름섬에서 유리병을 찾아 내용을 확인하세요.
② 유리병 근처에서 부싯돌을 챙기세요.🔍 획득 위치 인게임 확인
③ 보트 인근 모닥불에 부싯돌로 불을 붙이세요.
④ 비 오는 날 밤에 해변에서 탈옥수 NPC를 만나세요.🔍 날씨·시간 조건 인게임 확인
⑤ 암호 선택지가 나오면 '다루가'를 고르세요. 클리어!

함정 — 암호 선택지

⚠️ 헷갈리면 안 되는 포인트
암호 선택지에서 반드시 '다루가'를 고르세요. 다른 선택지를 고르면 클리어가 안 됩니다.
📷스샷 4: 유리병 발견 위치 + 암호 선택지 화면
📸 직접 촬영 권장 — 유리병 오브젝트 위치와 암호 선택지가 보이는 UI. 참고: 인벤 괴담집 공략

암호는 '다루가' — 오타 없이 정확하게

5. 위대한 제리 - 단계가 제일 긴 것

한 줄 조건 — 밤에 NPC 제리 2회 대화 → 시간 조작 다음날 → NPC 케이에게 녹슨 칼 획득 → 모닥불에 바치기

7개 중 단계가 제일 복잡한 편이라 순서대로 따라가시면 됩니다.

① 밤 시간으로 바꿔주세요. 주간에는 NPC 제리가 트리거가 안 됩니다.
② NPC '제리'를 찾아 대화하는데, 2회 해야 합니다.🔍 정확한 대화 횟수 인게임 확인 첫 번째 대화에서 키워드가 '낚시대회'와 '녹슨칼' 관련 내용이 나오니 눈여겨보세요.
③ 대화 후 시간 조작으로 다음날로 넘기세요. 같은 날에 계속 있으면 다음 단계가 안 열립니다.
④ 다음날이 되면 NPC '케이'를 찾아가 '녹슨 칼'을 받아오세요.
⑤ '녹슨 칼'을 들고 모닥불에 바치면 클리어입니다.

📷스샷 5: NPC 제리 2회 대화 화면 + 모닥불 바치기 화면
📸 직접 촬영 권장 — 제리 NPC 위치·대화창 한 컷, 모닥불 인터랙션 한 컷. 참고: 인벤 괴담집 공략

제리 2회 → 시간 조작 → 케이에게 녹슨 칼 → 모닥불 순서

6. 황금 나무의 비밀

한 줄 조건 — 밤에 빛나는 황금 나무 3곳 탐색 → 아이템 무덤에 바치기 → 무덤 속 검 활성화

밤에만 발광하는 황금 나무가 해오름섬에 3곳 있습니다. 밤 시간으로 맞추고 3곳을 모두 탐색하면 각각 아이템이 생성되는데요. 이 아이템들을 들고 무덤에 바치면 무덤 속에 잠들어 있던 검이 활성화되면서 괴담집이 클리어됩니다.

순서 정리하면:

① 시간 조작으로 밤으로 전환
② 빛나는 황금 나무 3곳 전부 탐색 (지도상 위치는 인게임에서 빛을 따라가면 찾을 수 있어요🔍 정확한 위치 인게임 확인)
③ 수집한 아이템을 무덤으로 가져가 바치기
④ 검이 활성화되면 클리어

📷스샷 6: 밤에 빛나는 황금 나무 + 무덤에 바치는 화면
📸 직접 촬영 권장 — 황금 나무가 발광하는 밤 장면과 무덤 인터랙션 창. 참고: 인벤 괴담집 공략

밤에만 빛나는 황금 나무 3곳 탐색 후 무덤에 바치기

7. 댄스 댄스 댄스!

한 줄 조건 — 아이템창 '큐박스 음악 플레이어' 등록 + 음악 켠 상태로 밤에 특정 장소 입장

인벤토리(아이템창)에 '큐박스 음악 플레이어'가 있어야 합니다. 이 아이템을 등록해 음악을 재생한 채로 밤 시간에 특정 위치에 들어가면 클리어가 됩니다. '큐박스 음악 플레이어'는 해오름섬에서 획득 가능한 아이템이니 미리 챙겨두세요.🔍 정확한 아이템 획득 방법 인게임 확인

순서:

① 인벤토리에서 '큐박스 음악 플레이어' 확인ㆍ등록
② 음악 재생 상태 유지
③ 밤 시간에 해당 장소(댄스 장소로 추정)에 입장
④ 클리어

📷스샷 7: '큐박스 음악 플레이어' 아이템창 등록 화면
📸 직접 촬영 권장 — 아이템 등록 UI + 밤에 해당 장소 입장 순간. 참고: 인벤 괴담집 공략

'큐박스 음악 플레이어' 켜고 밤에 입장하면 클리어

8. 나머지 2개 — 주인장도 직접 확인 중

괴담집 9개 중 7개는 위에서 정리해 드렸고, 나머지 2개는 현재 주인장이 직접 인게임 확인 중입니다. 아카라이브 이환 채널에서 '칼ㆍ검 위치 3곳' 관련 괴담집 언급이 보이긴 하는데, 정확한 이름ㆍ조건 텍스트는 직접 눈으로 확인 후 추가로 포스팅할게요. 확인되면 바로 댓글이나 추가 글로 알려드리겠습니다.

혹시 먼저 발견하신 분들 계시면 댓글로 알려주셔도 좋습니다(ㅋㅋ).

괴담집 수집이 의미 있는 이유가 하나 더 있는데요. 해오름섬 여행기 이벤트가 진행 중이거든요.
